## 개요

아하오호 갤러리는 창작물을 올리는 공간이지만, 현재 갤러리에서는 서로 소통할 수 있는 방법이 없다. 댓글 기능은 커뮤니티 영역에서만 제공되기 때문이다.

체험단은 운영자가 직접 선발한 멤버들로 구성된 안전한 환경이다. 이 환경을 활용해 갤러리 댓글 기능을 체험단 한정으로 먼저 열어두고, 아이들끼리 작품에 댓글을 주고받는 문화를 만들어본다. 동시에 아이들이 실제로 어떻게 소통하는지 관찰하며 추후 정식 댓글 기능 구상에 인사이트를 얻는다.

---

## 문제
갤러리에서 창작물을 올려도 서로 반응하고 소통할 수단이 없어 갤러리가 단방향 공간에 머물고 있음

## 우리의 해결책
크루(crew_status = active 또는 completed) 한정으로 갤러리 게시물 댓글 기능 오픈. 체험단이라는 제어된 환경에서 댓글 문화를 실험하고, 정식 기능 출시 전 실제 사용 패턴을 확인한다
커뮤니티 댓글과 동일한 컴포넌트·API 재사용. 추천 리액션은 도배 방지를 위해 비노출

## 예외 처리
- `crew_status = "dropped"`(탈락) 또는 크루가 아닌 경우 세션 기준으로 댓글 입력창 즉시 비활성화
- 기수 종료 후에도 댓글 작성 가능. 정식 댓글 기능 출시 전까지 유지, 출시 시 해당 정책으로 대체

## 원칙
1. 기존 커뮤니티 댓글 컴포넌트 재사용 — 신규 개발 최소화
2. 체험단 종료 후 정식 댓글 기능으로 자연스럽게 대체될 수 있도록 설계
