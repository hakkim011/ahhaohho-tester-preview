# 프로젝트 관리 어드민 기획

> 작성자: 김하경(하디)
> 작성일: 2026-06-16

---

## 기획 배경

현재 어드민에 사용자 게시물을 조회하거나 운영 처리할 수 있는 기능이 없음. 갤러리 운영과 피처드 프로젝트 확보라는 두 가지 목표를 위해 필요

## 기획 의도

운영자가 앱 내 게시물을 직접 조회하고 관리할 수 있도록 하기 위함. 부적절한 게시물에 즉각 대응하고, 챌린지별로 피처드 프로젝트를 지정·관리하여 체험단 운영 목표(챌린지별 피처드 확보) 달성 여부를 추적할 수 있도록 함. 자율게시 피처드는 이번 개발 범위 아님 — 추후 앱 내 아이디어 보드 기능과 연결 예정

---

## 기능 요약

### 게시물 조회

- 챌린지 / 자율게시 탭 필터
- 커뮤니티 게시물은 어드민 운영 대상 제외 — 커뮤니티 웹에서 선생님 권한으로 별도 관리
- 공개 게시물만 조회 — 나만보기 게시물 제외
- 최신순 / 오래된순 정렬
- 게시물 수 요약 표시 — "총 N개 · 피처드 N개" (현재 필터 기준)
- "피처드만 보기" 필터 — 체크 시 피처드 지정 게시물만 노출
- 게시물 카드에 본문 일부 표시 (2줄 말줄임), 카드 본문 영역에 1초 이상 호버 시 전체 본문 툴팁으로 표시
- 게시물 카드에 작성자 체험단 여부 배지 표시 — "체험단 N기" / "일반 유저"

### 챌린지별 피처드 선정 보드 (챌린지 탭 전용)

챌린지 탭 선택 시 게시물 그리드 상단에 챌린지별 피처드 확보 현황 테이블 노출. 체험단 운영 목표(챌린지별 피처드 확보) 달성 여부를 한눈에 추적하기 위함.

- 컬럼: # / 챌린지명 / 게시물 수 / 피처드 수 / 피처드 현황
- 피처드 현황 배지
  - ✅ 확보 (N) — 해당 챌린지에 피처드 1개 이상
  - ⚠️ 미확보 — 피처드 0개 (행 배경 강조로 시각적 구분)
- 챌린지 이름 검색창 — 입력 시 테이블 실시간 필터
- 행 클릭 시 해당 챌린지로 게시물 목록 필터 토글 (재클릭 시 해제)

### 게시물 운영

- 숨김 처리 — 사유 선택 + 인앱 알림 발송 (선택, 기본값: 미체크)
- 노출 전환 — 숨김 → 정상으로 복구
- 삭제 — 하드 삭제 (복구 불가) + 인앱 알림 발송 (선택, 기본값: 미체크)
- 숨김 상태 카드는 썸네일/내용 반투명 처리, 버튼은 선명 유지

### 피처드 프로젝트 지정

- 게시물 타입에 따라 유형 자동 결정
  - 챌린지 게시물 → 챌린지 피처드
  - 자율게시 → 자율게시 피처드 (이번 개발 범위 아님 — 추후 아이디어 보드 노출용)
- 피처드 지정 시 해당 챌린지 확인
- 피처드 해제 가능
- 숨김 상태 게시물은 피처드 지정 불가
- 피처드 지정 개수 제한 없음
- 앱에서는 피처드 풀 전체 중 최대 10개를 랜덤으로 노출 (어드민 지정 개수와 무관)
- 피처드 해제 후 재지정 가능

**피처드 지정 UX 상세**

1. "피처드 지정" 버튼 탭 → 확인 모달 진입
   - 게시물 정보 (챌린지명 + 작성자 + 날짜) 표시
   - 유형 자동 안내: "챌린지 게시물이므로 챌린지 피처드로 지정됩니다." / "자율 게시물이므로 자율 게시 피처드로 지정됩니다."
   - 챌린지 게시물인 경우에만 챌린지 선택 드롭다운 노출 (해당 챌린지 자동 프리셀렉트)
   - "취소" / "지정하기" 버튼
   - "지정하기" 시 선정 알림 자동 발송 (어드민 화면에 알림 문구·미리보기 미노출 / 문구 상세는 화면 설계서 참조)

2. "지정하기" 탭 시
   - 피처드 상태 변경 및 선정 알림 자동 발송
   - 토스트: "○○○의 게시물을 피처드로 지정했어요"

**피처드 해제 UX 상세**

- "피처드 해제" 버튼 탭 시 확인 모달 없이 즉시 해제
- 토스트: "○○○의 게시물 피처드 해제했어요"
- 해제 시 별도 알림 미발송

## 필요 API

| 메서드 | 경로 | 설명 |
|--------|------|------|
| GET | `/api/posts?type=challenge&challengeId=...&sort=desc&visibility=public` | 게시물 목록 조회 (나만보기 제외) |
| GET | `/api/challenges/:id/featured` | 챌린지 피처드 프로젝트 목록 조회 (앱 → 챌린지 상세 진입 시, 피처드 풀에서 최대 10개 랜덤 반환) |
| PATCH | `/api/posts/:id/hide` | 숨김 처리 |
| PATCH | `/api/posts/:id/show` | 노출 전환 |
| DELETE | `/api/posts/:id` | 삭제 (하드) |
| POST | `/api/posts/:id/featured` | 피처드 지정 |
| DELETE | `/api/posts/:id/featured` | 피처드 해제 |

## 참고 사항

- (미완료) 어드민에서 앱 사용자 게시물 조회 API — 제공 가능 범위 확인 필요
- 피처드 지정된 게시물을 유저가 직접 삭제할 경우 → 피처드 자동 해제, 챌린지 피처드 카운트 차감
- 피처드 지정된 게시물을 어드민에서 숨김 처리할 경우 → 피처드 자동 해제, 챌린지 피처드 카운트 차감
- 피처드 게시물 작성자가 탈퇴할 경우 → 게시물 비노출 처리와 함께 피처드 자동 해제
